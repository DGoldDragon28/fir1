The {\ttfamily wave-\/correlate} tool provides facility to compute standard deviation, covariance, pearsons correlation coeffiecient and SNR for input wav files.\hypertarget{md_correlate_README_autotoc_md32}{}\doxysubsection{Build}\label{md_correlate_README_autotoc_md32}
To try out {\ttfamily wav-\/correlate}, you can build the program with {\ttfamily cmake} from the root level {\ttfamily fir/} directory or from this directory as shown\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{cmake .}
\DoxyCodeLine{make}

\end{DoxyCode}
\hypertarget{md_correlate_README_autotoc_md33}{}\doxysubsection{Usage}\label{md_correlate_README_autotoc_md33}
Example usage is shown\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{Usage: wav-\/correlate <file1> [<file2>]}

\end{DoxyCode}
